## 🖥️ Prioridad Crítica: Migración a Máquina Independiente

### Estado Actual
- Ollama nativo en Mac Mini host
- PostgreSQL + Docker en volúmenes locales
- **Problema:** Imposible crecer sin máquina dedicada

### Plan de Migración (Mes 3-4)

**Opción 1: VPS/Cloud (Recomendado)**
- Hetzner, DigitalOcean, AWS EC2
- Costo: $50-150/mes
- Setup: Docker Compose + PostgreSQL + Ollama

**Opción 2: Mac Mini Remoto**
- M4/M3 + colocation/nube
- Costo: $50-100/mes
- Mejor rendimiento

**Cronograma:**
```
Mes 2   → Evaluar + Provisionar ($0 capex, $50-150 opex)
Mes 3   → Migration beta + testing
Mes 4   → Cutover completo a máquina independiente
Mes 5+  → Escalado sin límite de CPU/RAM
```
